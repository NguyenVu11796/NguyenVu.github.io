Feedback Analysis

Basically, my application raises question of adaptability and ethics: If a user changes their style over time, can the software keep up. Additionally, if my software can import data from other relevant sites, what about user privacy. My answer is to these questions is as follow: I will add an option where the user can take the survey and modify their personal profile again. My application will then personalize the learning interface accordingly. By doing this, the user can change how they want to absorb information whenever they like over time. As for the matter of privacy, my application will ask the user for their written consent (like those terms and conditions one sees when one downloads an application) but in a much more concise manner so that the user won’t skip the terms and conditions that follow. Moreover, even if the user refuses to let the application import the relevant data, the built-in survey is sufficient to build a customized interface.
