Installing a version control system and setting its pathway between our local machine and the GitHub repositories can be a pain in the neck. Luckily... follow these steps by presing "learn more" and it won't be so painful!

Step 7: setting up your GitHub and your local repository:
Type in the following commands: 
cd Desktop (if you want to work with your local desktop files instead of online clouds
git config ( you only have to do this once per PC or laptop.
git config --global user.name "your GitHub username" (setting up your path to GitHub username)
git config --global user.email "the email you used to sign in GitHub" (setting up your path to GitHub email)
git clone URL ( do it once to clone the entire online repositories from GitHub)
